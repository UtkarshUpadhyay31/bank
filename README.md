# 🏦 Bhaukali Bank – Flask Banking Web App

A modern, premium-looking banking web application built using **Python** and **Flask**.  
This project demonstrates full-stack fundamentals with backend logic, routing, and a stylish glassmorphism UI — all inside a single-file architecture.

---

## 🚀 Features

- 🆕 Account Creation with unique account number generation
- 🔐 Secure 4-digit PIN authentication
- 💰 Deposit functionality
- 💸 Withdrawal functionality
- 📄 Account detail viewing
- 🗄 JSON-based local database
- 🎨 Premium Dark Glass UI
- 📱 Responsive layout
- ⚡ Smooth animations and hover effects

---

## 🛠 Tech Stack

- **Python 3**
- **Flask**
- **HTML + CSS (Embedded UI)**
- **JSON (Database Storage)**

---

## 📂 Project Structure

```
app.py         # Main Flask application (routes, logic, UI)
data.json      # Local JSON database for account storage
main.py        # (Optional) Entry point or additional logic
```

---

## 🏁 Getting Started

1. **Install dependencies:**
   ```bash
   pip install flask
   ```
2. **Run the app:**
   ```bash
   python app.py
   ```
3. **Open in browser:**
   Visit [http://localhost:5000](http://localhost:5000)
